## data_types

### tuple
- 슬라이싱이 가능하다.
- 변경은 불가하다.
- 다양한 데이터 타입을 묶을 수 있다.
-   - (int,string,float)과 같은 형태도 가능

### list
- 슬라이싱이 가능하다.
- 변경또한 가능하다.
- 다양한 데이터 타입을 묶을 수 있다.
-   - (int,string,float)과 같은 형태도 가능

### set
- 슬라이싱이 가능하다.
- 변경또한 가능하다.
- 다양한 데이터 타입을 묶을 수 있다.
-   - (int,string,float)과 같은 형태도 가능
